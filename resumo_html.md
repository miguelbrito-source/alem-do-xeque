## O que é HTML?
O **HTML** (*HyperText Markup Language*) é uma **linguagem de marcação** fundamental para a web. É importante destacar que ele **não é uma linguagem de programação**, pois seu objetivo é estruturar e organizar o conteúdo (como um esqueleto), e não executar lógicas complexas.

## Estrutura Inicial Obrigatória
Para que uma página funcione, ela precisa seguir esta hierarquia básica:
1. `<!DOCTYPE html>`: Declaração para o navegador entender que usamos HTML5.
2. `<html>`: O elemento principal (pai) de toda a página.
3. `<head>`: Contém metadados, links de estilo e o título da aba.
4. `<body>`: Onde fica todo o conteúdo visível para quem acessa o site.

## Glossário de Tags Principais
* `<h1>` a `<h6>`: Cabeçalhos de títulos (o 1 é o maior, o 6 é o menor).
* `<p>`: Define um parágrafo de texto.
* `<a>`: Cria links (utiliza o atributo `href` para o destino).
* `<img>`: Insere imagens (utiliza `src` para o caminho e `alt` para acessibilidade).
* `<ul>` e `<li>`: Cria listas com marcadores (não ordenadas).

## Organização com a Tag <div>
A tag `<div>` funciona como um **container ou caixa**. Ela é essencial para o **aninhamento**, permitindo agrupar vários elementos em um só bloco. Isso facilita muito a organização do código e a aplicação de estilos (CSS) em seções específicas da página.
