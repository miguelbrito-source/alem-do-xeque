# AlémDoXeque

AlémDoXeque é um projeto educacional para ensinar e aprofundar o xadrez de forma clara e prática. O repositório aborda regras básicas, aberturas, meio-jogo, finais, princípios estratégicos, táticas, dicas e exercícios comentados, ajudando jogadores a evoluírem além do simples dar xeque.
